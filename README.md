# AI Data Analyst

An AI-powered data analysis platform built using FastAPI, Pandas, LangChain, Google Gemini, PostgreSQL, and AI Agents.

## Features

- User Authentication (JWT)
- Dataset Upload
- Automatic Metadata Extraction
- AI Data Cleaning Agent
- AI Business Understanding Agent (Coming Soon)
- AI SQL Agent (Coming Soon)
- Visualization Agent (Coming Soon)

## Tech Stack

- Python
- FastAPI
- PostgreSQL
- SQLAlchemy
- Alembic
- Pandas
- Google Gemini
- JWT Authentication
